## Cmd AI
A terminal based AI client with a bad name

It was made to find out that is making a DIY Claude Code is cheaper than the actual

## How to start
**You must have:**
- pnpm
- node.js

**.env**
You must have your Claude api key in the .env file like this:
```
API=sk...
```

**Install thing**
Run
```
pnpm i
```
to install the required packages

**Start**
To start run
```
node .
```

## Usage

To use you can ask it a question. It logs what it is doing in the terminal

When it tells you to press enter then press it to continue
